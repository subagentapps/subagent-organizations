## README.md

```markdown
# agentknowledgeworkers.com

Landing page and live project tracker for Agent Knowledge Workers. Dogfoods
[knowledge-work-plugins-cli](https://github.com/agentknowledgeworkers/knowledge-work-plugins-cli) ŌĆö
every milestone in kwpc drives progress shown on this site.

## Stack

- **Vite 6** + **React 19** + **TypeScript**
- **Cloudflare Pages** (hosting + edge functions)
- **lucide-react** (icons)
- No runtime CSS framework ŌĆö design tokens in `src/App.tsx`

## Local dev

```bash
nvm use           # Node 20
npm install
npm run dev       # http://localhost:5173
```

## Deploy

Merge to `main`. GitHub Actions builds and deploys to Cloudflare Pages.
Manual deploy: `npm run build && npm run deploy`.

## Structure

- `src/App.tsx` ŌĆö dashboard (to be split into `components/`)
- `functions/` ŌĆö Cloudflare Pages Functions (GraphQL proxy to GitHub, coming soon)
- `.github/workflows/deploy.yml` ŌĆö CI/CD
```
